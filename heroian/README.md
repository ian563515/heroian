# 神龍英雄科技部落格後台管理系統

## 系統介紹

神龍英雄科技部落格後台管理系統是一個功能強大的內容管理平台，專為管理部落格文章、聯絡資訊和系統設定而設計。系統採用現代化的技術架構，提供直觀易用的用戶界面。

## 主要功能

### 1. 用戶認證與安全
- 安全的登入系統，支持密碼加密存儲
- "記住我"功能，方便管理員快速登入
- 完整的會話管理和安全登出功能

### 2. 文章管理
- 新增、編輯和刪除文章
- 文章分類管理
- 文章狀態管理（草稿/已發布）
- 文章統計功能

### 3. 聯絡資訊管理
- 查看來自網站的聯絡資訊
- 標記聯絡資訊為已讀/未讀
- 回覆訪問者資訊
- 聯絡資訊統計

### 4. 系統設定
- 網站基本設定管理
- 管理員個人資料和密碼修改
- 系統狀態監控

### 5. 數據統計
- 文章數量統計
- 聯絡資訊統計
- 系統運行狀態監控

## 安裝與配置

### 系統需求
- PHP 7.0 或更高版本
- 支持 PDO 和 SQLite 擴展
- 現代網頁瀏覽器

### 安裝步驟
1. 將所有文件上傳到您的網站伺服器
2. 確保目錄權限正確（通常需要讀寫權限）
3. 在瀏覽器中訪問 `src/admin/create_admin.php` 初始化管理員帳號
4. 使用默認管理員帳號登入：
   - 用戶名：admin
   - 密碼：admin123
5. 登入成功後，刪除 `src/admin/create_admin.php` 文件以確保安全

### 數據庫配置
系統使用 SQLite 數據庫，數據庫文件會自動在 `data` 目錄中創建。如需手動配置，請編輯 `src/admin/config/database.php` 文件。

## 默認管理員帳號

系統初始化時會自動創建以下管理員帳號：
- 用戶名：admin
- 密碼：admin123

**安全提醒**：首次登入後，請務必修改默認密碼！

## 使用指南

### 登入系統
1. 訪問 `src/admin/login.html` 進入登入頁面
2. 輸入用戶名和密碼
3. 可選擇"記住我"功能
4. 點擊"登入"按鈕

### 文章管理
1. 登入後台，點擊側邊欄的"文章管理"
2. 點擊"新增文章"按鈕創建新文章
3. 填寫文章標題、摘要、內容等資訊
4. 選擇文章分類和狀態
5. 點擊"儲存"按鈕保存文章

### 聯絡資訊管理
1. 點擊側邊欄的"聯絡資訊"
2. 查看所有訪問者提交的聯絡資訊
3. 點擊"查看"按鈕查看詳細資訊
4. 可選擇標記為已讀、回覆或刪除

### 系統設定
1. 點擊側邊欄的"系統設定"
2. 修改網站標題、描述等資訊
3. 點擊"儲存變更"按鈕保存設定

### 管理員設定
1. 點擊側邊欄的"管理員設定"
2. 修改個人資料和密碼
3. 點擊"儲存變更"或"修改密碼"按鈕

## 安全建議

1. 首次登入後，立即修改默認密碼
2. 定期更換管理員密碼
3. 不要在公共電腦上使用"記住我"功能
4. 確保伺服器軟體和系統保持最新
5. 刪除不必要的文件（如 `src/admin/create_admin.php`）
6. 定期備份數據庫文件

## 常見問題

### Q: 忘記管理員密碼怎麼辦？
A: 可以直接刪除數據庫文件，系統會自動重新創建一個新的管理員帳號，但這樣會導致所有數據丟失。建議聯繫技術支援。

### Q: 如何備份數據？
A: 直接複製 `data` 目錄即可備份所有數據。還原時，只需將備份文件複製回 `data` 目錄。

### Q: 如何添加新的管理員？
A: 目前系統僅支持單一管理員帳號。如需添加多管理員，需要修改數據庫結構和相關代碼。
